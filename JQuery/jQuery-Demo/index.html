Manipulation (retrieving or setting value or text in any HTML element) .after() .append() .prepend() .attr() .before() .html() .text() .val() Events (functions to handle an event) .click() .on() .hover() Note that for the manipulation jQuery functions, there are few distinctions to make. The functions .html(), .text(), .append() are different in a key way: .html() can be used to get or replace with HTML markup inside, meaning new HTML tags. (Ex. replacing any existing elements inside div by paragraph) .text() is used to get or set just the text value of an HTML element. (Ex. text of a paragraph) .append() will add markup/tag inside the specific element (Ex. adding list item in ordered list)
